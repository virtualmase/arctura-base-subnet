# Funding Guide — Base Programs

Four programs available. Stack them sequentially.

## 1. Base Builder Rewards (Start Now)
- **Amount:** Up to 2 ETH/week from a shared pool
- **Requirement:** Build on Base, document publicly on X weekly
- **Apply:** [builderscore.xyz](https://builderscore.xyz)
- **When:** Week 1 — apply immediately when repo goes live

## 2. Base Builder Grants (Phase 1 → 2)
- **Amount:** 1–5 ETH retroactive
- **Requirement:** Shipped prototype deployed on Base or Base Sepolia
- **Apply:** [paragraph.com/@grants.base.eth](https://paragraph.com/@grants.base.eth/calling-based-builders)
- **When:** Once miner.py + validator.py run on testnet

## 3. OP Retro Funding (Ongoing)
- **Amount:** Variable, based on public goods impact
- **Requirement:** Open source, measurable public benefit
- **Apply:** [retrofunding.optimism.io](https://retrofunding.optimism.io) via Atlas
- **When:** Track contributions from day 1. Apply retroactively.
- **Qualifies:** Subnet code, docs, training dataset, subnet agent (all Apache-2.0)

## 4. Base Batches (Phase 2 → 3)
- **Amount:** Significant funding + Coinbase Ventures pitch day
- **Track:** Buildathon → 4-week Incubator → VC pitch
- **Apply:** [basebatches.xyz](https://basebatches.xyz)
- **When:** Phase 2, with live testnet netuid as MVP proof
- **Frame:** "First Base subnet on Bittensor — live testnet, open source, funded by Base"
